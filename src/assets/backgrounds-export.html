Cómo exportar:
1. Abre este archivo en Chrome o Safari.
2. Para cada fondo, haz clic derecho → "Inspeccionar", selecciona el elemento .slide y en DevTools → ⋮ → "Capture node screenshot" → descarga el PNG exacto de 1080×1080.
3. O también puedes usar la extensión GoFullPage o Cmd+Shift+4 y seleccionar exactamente el cuadrado.
